# react-native-armfit-sdk

connect and retrieve data from BLE devices

## Installation

```sh
npm install react-native-armfit-sdk
```

## Usage

```js
import { multiply } from "react-native-armfit-sdk";

// ...

const result = await multiply(3, 7);
```
